# yata-technologies Homebrew Tap

Homebrew tap for YATA TECHNOLOGIES apps.

## Folda 🦉

Dual-pane macOS file manager.

```sh
brew install --cask yata-technologies/apps/folda
```

(Equivalent to `brew tap yata-technologies/apps && brew install --cask folda`.)

The app is Developer ID signed and notarized, so it opens without Gatekeeper warnings.
